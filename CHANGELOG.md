## [1.0.0] - 2026-05-15

### Added
- Initial public release of SecureVault desktop app.
- Account type template management (custom fields per type).
- Expiration support for accounts with visual expiry warnings.
- Portable mode support and installer build workflow.
- Import dry-run validation before restore overwrite.
- DB health check integration and sidebar warning.
- Metadata hardening migration for sensitive fields.
- Rust automated tests for crypto, metadata compatibility, and expiry persistence.

### Changed
- Improved account search to include fields, tags, and customer context.
- Improved Vietnamese labels across key screens.

### Security
- Encrypted sensitive metadata and migration path for legacy plaintext metadata.
